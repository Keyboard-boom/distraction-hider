# Distraction Hider

一个 Chrome MV3 扩展，用来像 Safari 的“隐藏干扰项目”一样，在网页上手动选择并隐藏不想看的元素。

## 功能

- 点击扩展图标后选择“选择隐藏”，在页面中点选某个元素并隐藏。
- 选择“隐藏相似”，尽量隐藏同类列表项、广告卡片或重复模块。
- 每个站点单独保存规则，刷新页面后自动生效。
- 支持撤销上一条规则、删除单条规则、恢复当前站点、暂停或启用当前站点规则。
- 规则保存在 `chrome.storage.sync`，没有远程服务，也不会上传浏览数据。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择这个目录：`/Users/jh/Documents/New project/distraction-hider`。

## 使用

1. 打开任意网页。
2. 点击工具栏中的 Distraction Hider。
3. 点“选择隐藏”后，在网页上点击要隐藏的项目。
4. 需要恢复时，再打开弹窗撤销、删除规则或恢复当前站点。

Chrome 不允许扩展修改 `chrome://`、Chrome Web Store 等受保护页面，这是浏览器限制。

如果要在本地 `file://` 页面使用，需要在扩展详情页额外开启“允许访问文件网址”。
